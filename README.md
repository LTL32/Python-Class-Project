# Python-Class-Project
A Pyhton Program for our class
## Description, Levar
This program asks the user to enter their name and three exam scores. 
It calculates the average of the three scores and determines the corresponding letter grade (A, B, C, D, or F). 
The program then displays a formatted grade report showing the student's name, average score, and final letter grade.

## How to Run, Darrin
1. Open the Jupyter Notebook file.
2. Run all cells.
3. Enter the requested inputs when prompted.
